## Babel v0.1.0 — pre-alpha

First public build. Everything here runs on-device; nothing leaves your Mac.

### What's in the box

- **Push-to-hold dictation** — hold Right Option, speak, release. The text lands in whatever app you're focused on.
- **Three modes, all local**
  - *Fast* / *Balanced* — Apple SpeechAnalyzer (macOS 26's new on-device transcription)
  - *Accurate* — Whisper `large-v3-turbo` via [WhisperKit](https://github.com/argmaxinc/WhisperKit); ~1.5 GB model downloaded on first use from Hugging Face
- **Liquid Glass pill** with an audio-reactive blob, rendered natively on macOS 26
- **Hybrid text insertion** — Accessibility API first, pasteboard + ⌘V as fallback, raw keystroke synthesis for terminals (Terminal.app, iTerm2, Ghostty, Kitty, Alacritty, Warp, Hyper)
- **Multi-language dictation** — picks up whatever locales you've enabled in macOS Keyboard settings; language selector in Settings → General
- **Optional polish** — start/stop sounds, launch at login, permission-status banners
- **History** — every dictation kept locally in SwiftData; copy or delete from the history window
- **Secure-input awareness** — if Terminal has Secure Keyboard Entry on, the transcript is left on the clipboard and the pill says "On clipboard — ⌘V"

### Install

1. Download `Babel-0.1.0.dmg` below.
2. Open the DMG and drag **Babel.app** to **Applications**.
3. **Right-click → Open** the first time (not double-click). macOS will ask once whether to open an app from an unidentified developer; allow it. After that, regular double-click works.
4. First launch walks you through four permissions: Microphone, Speech Recognition, Input Monitoring, Accessibility.

The right-click-Open dance is because this release is signed with a free Apple Personal Team certificate — enough for Gatekeeper to recognize the signature but not to skip the unidentified-developer prompt. A notarized build is [on the roadmap](https://github.com/matinsaurralde/babel/blob/main/docs/release.md).

### Requires

- macOS 26 Tahoe (26.0+)
- Apple Silicon
- About 2 GB free disk if you use Accurate mode (Whisper model download)

### Known limits

- **Hotkey is hardcoded** to Right Option. Rebinding is the next feature.
- **Accurate mode first run** takes 30–60 s the first time (model download + CoreML graph compilation). Subsequent sessions are fast.
- **Pill doesn't show live partials** — by design; transcript lands on key release.
- **Terminal.app with Secure Keyboard Entry on** can't receive insertion — transcript stays on the clipboard, you ⌘V manually.
